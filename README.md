# SPD

> Minecraft 1.20.1 的渊蚀探索、生态威胁与炽脉锻造模组。支持 Forge 与 Fabric。

SPD 将主世界的一部分地貌转化为被渊浊石、异化菌丝与血黑水侵染的区域。玩家需要在高风险生态中采集炽脉与血烬资源，建立加工设施，制作能应对感染生物的武器与装备。

当前工程版本为 `1.0-SNAPSHOT`。数值与具体掉落会随开发迭代调整，游戏内提示和 JEI 配方为当前版本的最终依据。

## 核心玩法

1. 探索主世界，寻找炽脉石、血烬矿和渊蚀地貌。
2. 将炽脉原矿放入高炉处理为炽脉碎片。
3. 在渊心熔铸台中用炽脉碎片与燃料加工炽炭钢锭。
4. 用炽炭钢锭、烬柄和其他感染材料制作工具、近战武器与功能性方块。
5. 在渊蚀环境中管理「渊蚀压」，应对成群的异化生物与特殊天气。

## 渊蚀世界

SPD 新增四个可通过 `/locate biome` 查询的渊蚀群系：

| 群系 | 特征 |
| --- | --- |
| 渊蚀血漠 | 由渊浊血沙组成的大型荒漠，拥有枯木、炽脉资源与感染生物。 |
| 渊蚀海岸 | 血黑色海水环绕的海岸区域，是渊蚀海龟的活动地带。 |
| 菌蚀浅滩 | 带有血黑水色调的浅海生态，适合进行危险的水下探索。 |
| 铬质海底矿洞 | 海底洞穴地貌，炽脉矿物的生成显著提高。 |

常见环境方块包括渊浊石、渊浊血沙、蔓疫节点、广域疫变和渊蚀龟蛋。血沙会拖慢不属于 SPD 阵营的生物；感染区域则会持续施加侵蚀压力。

### 渊蚀压

「渊蚀压」是贯穿探索与战斗的叠层负面状态。渊蚀环境、感染生物的攻击以及孢雾都会使层数上升；脱离侵蚀源后会逐步衰减。

- 中高层数会降低战斗与移动能力，并造成持续侵蚀伤害。
- 满层侵蚀可将部分原版狐狸、狼和僵尸转化为相应的 SPD 变体。
- SPD 阵营生物不会受到渊蚀压影响。
- 净化相关物品、方块和抗性装备可用于降低风险。

## 异化生态

| 生物 | 定位与特征 |
| --- | --- |
| 霉染僵尸 | 受霉变驱动的敌对生物，部分个体会在接近目标时飞扑。 |
| 渊蚀蠹虫 | 高机动的小型敌对生物，会主动袭击非 SPD 生物。 |
| 渊蚀狐 | 中立偏敌对的掠食者，能快速突进、召集同类并掠取物品。 |
| 渊蚀狼 | 以小群体行动的追猎者，擅长群嚎支援、扑击与撕裂。 |
| 渊蚀海龟 | 具有重甲、防御孢雾、缩甲御守与水下冲撞能力的海岸生物。 |
| 假母体 | 感染扩散机制中的高生命实体，会在周围留下广域疫变并制造威胁。 |

这些生物会掉落菌蚀残渣、铬尘、渊心孢子、腐沙鳞皮、蚀纹龟甲片等材料。所有 SPD 生物死亡时还有机会掉落液金；玩家需要持有培养基才能拾取它。

## 炽脉锻造

炽脉资源线提供从矿物到武器的主要成长路径：

```text
炽脉石 -> 炽脉原矿 -> 炽脉碎片 -> 炽炭钢锭 -> 装备与武器
```

- **炽脉石**：主世界与铬质海底矿洞中的关键矿物。
- **血烬矿**：可烧制为血烬锭，再加工为烬柄。
- **渊心熔铸台**：将炽脉碎片加工为炽炭钢锭的核心装置，并可能产出副产物。
- **烬玄炉壁与熔铬注嘴**：用于搭建和操作多格炼炉结构。
- **培养基**：由玻璃、糖与水瓶加工而成；处理液金时不可缺少。
- **渊炽铭符碑**：两格高的功能性方块，配方由 4 个黑曜石围绕紫水晶碎片组成。

## 武器与装备

SPD 提供炽烬工具组，以及面向不同战斗节奏的专属近战武器：

- **炽脉巨剑**：以层数积累与范围爆发为核心的重型武器。
- **炽脉贯刺枪**：可右键蓄力，沿准星方向发动远距离穿刺。
- **炽脉疾锋刀**：快速连击会叠加「疾锋」，满层触发额外的瞬斩伤害。
- **无名之剑**：具备蓄力镇渊斩击，对渊蚀阵营有更强的压制能力。

武器能够造成或利用「灼脉」「撕裂」「侵蚀抑制」等状态，与渊蚀压和敌人的防御机制形成相互克制。

## 渊默天气

主世界可能遭遇「渊默」天气。天气期间，户外生物会受到风力影响，暴露在天空下的玩家会承受额外压力，区域内还可能形成渊默龙卷风。管理员可使用下列指令控制天气：

```text
/spd weather abyssal_gloom
/spd weather abyssal_gloom true
/spd weather abyssal_gloom false
/spd weather abyssal_gloom tornado <持续秒数>
```

SPD 还提供独立世界难度指令，可通过 `/spd difficulty` 或 `/dif world` 查看与切换。

## 安装

| 项目 | 要求 |
| --- | --- |
| Minecraft | `1.20.1` |
| Java | `21` 或更高版本 |
| 加载器 | Forge `47.4.x` 或 Fabric Loader `0.19.3+` |
| 外部前置 | Architectury API、GeckoLib、TerraBlender；Fabric 端还需要 Fabric API。 |

当前 Forge 与 Fabric 构建均通过 Jar-in-Jar 内嵌 **Photon** 与 **LDLib2**。正常安装 SPD 成品 JAR 时，无需再单独放入这两份前置，除非其他模组另有版本要求。

## 构建

本仓库使用 Architectury，公共玩法代码位于 `common/`，加载器入口分别位于 `fabric/` 与 `forge/`。

```powershell
gradle :forge:build --console=plain
gradle :fabric:build --console=plain
```

构建产物位于对应模块的 `build/libs/` 目录。

## 开发资源

`杂物/` 中保留了 Blockbench `.bbmodel` 等美术源文件。游戏实际加载的 GeckoLib 模型、纹理和动画资源位于 `common/src/main/resources`，需要调整实体表现时应修改或重新导出对应资源。
