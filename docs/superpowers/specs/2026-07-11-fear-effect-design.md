# 恐惧效果设计

## 目标

新增 `spd:fear` 负面状态效果，对玩家和其他生物统一生效。效果期间，受影响实体持续向自身面朝方向的反方向移动，并按效果等级降低攻击伤害。

## 等级与攻击力

Minecraft 效果 amplifier 从 0 开始，因此实际等级为 `amplifier + 1`。每级使攻击伤害降低 10%：恐惧 I 为 -10%，恐惧 II 为 -20%，依此类推。

攻击伤害使用 `Attributes.ATTACK_DAMAGE` 的乘算属性修饰符实现，使玩家和具备攻击属性的生物遵循同一规则。高等级效果的最终攻击伤害最低为 0，不允许因恐惧产生负攻击力。

## 强制倒退

效果每 tick 执行一次，根据实体当前水平朝向计算反方向速度。倒退只覆盖水平运动分量，保留当前垂直速度，因此不会取消跳跃、坠落、击飞的垂直运动。

移动由状态效果逻辑统一施加，对玩家和普通生物使用相同路径。实体乘坐载具、死亡或失去行动能力时不施加强制移动。服务端负责权威移动并标记速度变化，客户端玩家同步应用相同效果以减少输入抵消造成的抖动。
